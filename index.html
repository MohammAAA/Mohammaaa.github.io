IMT ARM Based Embedded Systems Diploma

In this diploma, we dig through the ARM architecture, investigate STM32F1 microcontroller, some of its core peripherals, bootloader implementation overview, try out some intersting applications such as playing sounds with DAC, performing IR communication, interfacing with TFT touch screen, connecting microcontroller to web server and communicating the controller over HTTP protocol.
